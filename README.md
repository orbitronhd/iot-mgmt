# IoT Device Management (Smart Home)

A full-stack, event-driven MERN application that utilizes WebSockets to bidirectionally synchronize a React control dashboard with a Node.js CLI simulator acting as virtual IoT devices.

## Prerequisites

* [Node.js](https://nodejs.org/)
* Required libraries

## How to run

```bash
git clone https://github.com/orbitronhd/iot-mgmt.git
cd iot-mgmt
npm run dev
